# lego

## 描述
vue3 jsx 实现可视化搭建平台

## 核心知识点
- vue3
- 拖拽API
    - drop
    - draggable
    - dragenter
    - dragover
    - dragleave
    - onDragstart
    - onDragend
- 鼠标事件API
    - onMousedown
    - mousemove
    - mouseup

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
